LN4 Proof by Induction


Ordinary Induction


To prove P(x) for all x ≥ n: Base case: compute P(n) directly and show that it is true. Induction step: assume P(m) is true and derive that P(m+1) is true.

Example: prove the Cauchy inequality (for arbitrary ai, bi) $$a_{1}b_{1} + a_{2}b_{2} + \cdots + a_{n}b_{n} ;\leq; \sqrt{a_{1}^{2} + a_{2}^{2} + \cdots + a_{n}^{2}} ; \sqrt{b_{1}^{2} + b_{2}^{2} + \cdots + b_{n}^{2}}\ \ \ \ \ (n)$$ $$Proof:\ Base\ case:|a_1b_1| = \sqrt{a_1^2}\sqrt{b_1^2},\ \ so\ a_1b_1 ≤ \sqrt{a_1^2}\sqrt{b_1^2}\ \ \ \ \ (1)$$ $$Induction:Given\ (n),\ denote\ it\ as\ \ x≤pq,\ so:$$ $$x+a_{n+1}b_{n+1}≤pq+a_{n+1}b_{n+1}$$ $$Given\ (2),\ pq+a_{n+1}b_{n+1}≤\sqrt{p^2+a_{n+1}^2}\sqrt{q^2+b_{n+1}^2}\ \ \ \ \ (n+1)\ \ Q.E.D$$ (Because the recursion implicitly uses the fact that the inequality holds for two terms, one must additionally verify the two-term case.)



Strong Induction


To prove P(x) for all x ≥ n: Base case: compute P(n) directly and show that it is true. Induction step: assume P(n), …, P(m) are all true, and derive that P(m+1) is true.

Example: prove that every integer greater than 8 can be formed using additions of only 3 and 5. List the first few cases: 3+5=8, 3+3+3=9, 5+5=10. For n = 11, we have n-3 = 8, which is representable. …… For n = m, if n-k3 = 8 or 9 or 10, then n is representable. (For any n ≥ 11, n-k3 must equal 8, 9, or 10 for some k, so n can be formed by adding k copies of 3.)



The Well-Ordering Principle (WOP)


$ WOP: every nonempty set of natural numbers has a least element. Equivalent form: every nonempty set of positive integers has a least element.

To prove P(x) for all x ∈ N: Assume ¬P(x), and let S ::= {n ∈ N | ¬P(n)}. By WOP, S has a least element. Then derive some property of this least element that forces the existence of an even smaller counterexample, yielding a contradiction.


Example 1: Prove that √2 is irrational


Proof by Contradiction: Suppose √2 is rational, then it can be written as √2 = p / q, where p and q are natural numbers. Then, in the natural set of denominators p, there must be a least one (By WOP). $${p^2 \over{q^2}}=2,\ so\ smallest\ p\ should\ be\ a\ even\ number:\ p_0=2k, q_0=2l$$ $$4k^2=2 imes 4l^2,\ but\ k={p_0\over 2}\ also\ hold,\ so\ p_0\ is\ NOT\ smallest.$$ There is always a smaller denominator, so the hypothesis is false.


Example 2: Prove that the equation 4a3+2b3=c3 has no positive integer solution


Suppose there is a Z+ solution for this function. then S::={a ∈ℤ+| ∃𝑏,𝑐∈ℤ+, 4a3+2b3=c3} must have a smallest element (By WOP). Denote smallest one as a0, b0, c0. $$Since\ 4a^3\ and\ 2b^3\ are\ both\ even,\ c^3\ and\ hence\ c\ are\ even,\ denote\ c_0=2p$$ $$Thus,\ 4a^3+2b^3=(2p)^3 o b^3=4p^3-2a^3,\ hence\ b^3\ and\ b\ are\ even,\ denote\ b_0=2q$$ $$a^3=2p^3-4q^3, then\ a_0,b_0\ and\ c_0\ are\ all\ even$$ $$However,\ ({a\over 2},{b\over 2},{c\over 2})\ also\ must\in S, thus\ hypothesis\ is\ false.$$



Invariant Method


Given a sequence of integers (say, the integers from 1 to 100), each move removes two numbers and inserts the sum of their squares. Is the final remaining number fixed? Will it be odd or even?

Key idea: the overall parity is an invariant. The magnitude is not fixed, because the returned sum of squares may itself be chosen again later. However, regardless of which two numbers are removed, the parity of the sum of their squares is always equal to the parity of their sum. Therefore, the parity of the final number must equal the parity of the sum of all original numbers, which is even.


Thought Problem


Conjecture: Take the configuration on the left as the initial state, namely an even permutation. Let the parity of the permutation, meaning the number of swaps needed to reach it from the initial state modulo 2, be denoted by a. Number the rows from bottom to top and the columns from right to left, and let the parity of the blank position be x⊕y:

The total parity is (x⊕y)⊕a, and in the initial state it is 0⊕0⊕0=0. Treat the blank as the moving object. If the blank swaps upward once, we get 1⊕0⊕1=0; if it then swaps left once, we get 1⊕1⊕0=0; and so on. No matter what happens, the result must remain 0. But the target arrangement on the right is an odd permutation, so 0⊕0⊕1=1. Hence it is impossible.
