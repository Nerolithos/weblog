CSC3200 LEC 1-2 Introduction to C++


1. How Do You Run It?


macOS comes with clang++ by default. You can check it in the terminal with:

clang++ --version

If it is unavailable, try brew install llvm or install g++ via Homebrew.

To compile a single-file program such as hello.cpp:

clang++ -std=c++17 -Wall -O2 #filename#.cpp -o #compilename#
clang++ #filename#.cpp -o #compilename#  // you may also compile this way; it is faster, but not ideal for important code
./#compilename#  // run

On Windows, one may use Dev-C++, or install a toolchain such as MSYS2 / MinGW for use with VS Code’s integrated terminal.


Multi-File Compilation

src/ math.h, math.cpp, main.cpp

clang++ -std=c++17 -Wall -O2 src/main.cpp src/math.cpp -Iinclude -o app
./app


2. Data Types, Operators, and Naming Rules


Data Types

using namespace std;
int age = 20;      // integer
double pi = 3.14;  // floating-point number
char grade = 'A';  // character
bool flag = true;  // Boolean value

int constant;  // allowed
const int constant;  // not allowed: a constant must be initialized at declaration time

Note that in C there is another common way to define constants: #define constant 42. But it is only a preprocessor substitution, not a real typed constant, and it may easily create inconsistencies across files.


Operators

int m = double(4) % 5;  // 4
double n = int(4.2) / 2;  // 2.0
double p = 4 / 5;  // 0.0
// so to obtain a fractional result, at least one operand in the expression should be of type `double` or `float`

#include <iomanip>
cout << setprecision(3) << 3.141592 << "/n";  // keep three decimal places

Increment operators (the same logic applies to decrement):

int a = 0;
int b = 0;
int x = a++;  // assign first, then increment: a=1, x=0
int y = ++b;  // increment first, then assign: b=1, y=1

Boolean Operators

-1 && double(2.0) == bool(true)  // every nonzero number is true; true && true is true
"0" || NULL != false  // nonempty character strings are true; true || false is true

Ternary operator:

cout << ((a % 2 == 0) ? "Even" : "Odd") << endl;  
// if the condition before ? is true, output the first option; otherwise output the second

The string type should be included explicitly. Sometimes iostream indirectly pulls it in, but one should not rely on that.

using namespace std;
#include <string>
string name = "csc3200"

To include a standard library header, write #include <...> — without a semicolon.

To include a local header file, write #include "....h" — again, no semicolon.

A .cpp source file is not included the same way a header is.


Naming Rules

  • Use only letters, digits, and underscores; identifiers cannot begin with a digit.
  • C++ is case-sensitive: Age and age are different variables.
  • Keywords such as int, if, and class cannot be used as identifiers.
  • Common naming styles include camelCase and snake_case.

Many standard-library symbols are stored in the std namespace. One may either declare

using namespace std;   // to import only one symbol, write: using std::string
#include <string>
string name = "csc3200";

Otherwise, one must write:

 std::string name = "csc3200";


3. Standard I/O


cout

  • short for “character output”; it denotes the standard output stream, usually the screen
  • provided by the header #include <iostream> Example:
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, world!";  // output to the screen
    return 0;
}

<< (stream insertion operator)

  • stream insertion operator
  • It inserts the content on the right into the output stream on the left.
  • Under all circumstances, << does not automatically insert spaces between output items. Example 1:
cout << "The answer is: " << 42;
  • the string "The answer is: " is inserted into cout and displayed on the screen.
  • then 42 is inserted into the same stream and displayed immediately after it.
  • output：
The answer is: 42

Example 2:

#include <string>
#include <iostream>
string s = "Hello";
s += " World";              // 拼接
cout << s << " 长度: " << s.length() << endl;
  • output：
Hello World 长度: 11

endl

  • short for “end line”
  • It creates a newline (similar to the escape sequence "\n") and flushes the output buffer.
  • provided by the header #include <iostream> Example:
#include <iostream>
cout << "1" << endl << "2"
cout << "3\n" << "4";
  • output：
1
2
3
4

Difference:

  • \n is only a newline escape sequence.
  • endl = newline + forced buffer flush (immediate display).
  • When output goes to a terminal, \n often appears to flush naturally. But if the output target is a file or a pipe, \n does not trigger flushing, so writing may wait until the buffer fills.

>> (stream extraction operator)

This is usually paired with cin (“character input”) to read information from the keyboard:

#include <iostream>
#include <string>
using namespace std;

int main() {
    int age;
    string name;

    cout << "请输入姓名和年龄: ";
    cin >> name >> age;   // 变量之间空格或换行皆可
    cout << "结果: " << name << " " << age << endl;
    return 0;
}

If you do not want input to stop at spaces, read the whole line directly:

string name;
getline(cin, name);  // 读到回车为止

However, if getline() comes immediately after another cin or getline, the leftover newline may be read as the next input:

cin >> age;  // 输入 18↵
cin.ignore(numeric_limits<streamsize>::max(), '\n');
getline(cin, name);  // 省略上一行的话，这里直接读到空行！


## 4. Function Definition and Invocation


  • main() is the entry function. It runs automatically and should not be called manually. Its return type is always an integer; returning 0 indicates normal termination.
  • A function may contain multiple return statements, but any single execution path hits only one of them. Every return must match the declared return type.

Example:

#include <iostream>
using namespace std;

// 计算差值
int diff(int a, int b) {
    if (a > b) return a - b;
    if (a <= b) return b - a;
}

int main() {
    cout << diff(3, 5) << endl;
    return 0;   // cpp-11 以后的版本中可以省略
}
  • General form:
返回类型 函数名(参数列表) {
    // 函数体
    return 值;
}


## 5. Conditional Control

int x = 10;
if (x > 0) {
    cout << "positive" << endl;
} else if (x == 0) {
    cout << "zero" << endl;
} else {
    cout << "negative" << endl;
}

switch:

switch (#变量或表达式#) {
    case1:
        #条件#;
        break;
    default:
        #条件#;
        break;
}


## 6. Loops


for Loop

for (int i = 0; i < 5; i++) {
    cout << i << " ";
}

Output: 0 1 2 3 4


while Loop

int i = 0;
while (i < 5) {
    cout << i << " ";
    i++;
}

do-while Loop

int i = 0;
do {
    cout << i << " ";
    i++;
} while (i < 5);

(executes at least once)



7. Addresses, Pointers, References, and Passing


&a: take the address of variable a.

int& r = a: create a reference to a (give a another name r).

int* p = &a: declare p as a pointer to a.

*p: access the value pointed to by p.

int a = 5;
int* p = &a;
int& r = a;  // r 引用 a
cout << *p;  // 输出 12 位 16 进制整数
*p = 20;  //指针也可以直接修改对应变量的值
p = &r;  //指针可以改变对象，但此时位置不变，因为 r 引用 a
cout << (&r == &a) << "\n";  // return 1 （C++ 里输出 true 会变成 1）

const int& r = a: constant reference (you cannot modify a through r).

int* const p = &a: constant pointer (you may modify the pointee, but cannot redirect the pointer).

const int* p = &a: pointer to const (you cannot modify the pointee, but the pointer may be redirected).

const int* const p = &x: const pointer to const.


  • Pass by pointer: modify a without copying it by passing pointer p; this is compatible with C.
 void setZero(int* p) { *p = 0; }
 setZero(&a); // a 变成 0
  • Pass by reference: modify a without copying it by passing alias b; this is not a C feature.
void SetZero(int& b) { b = 0; } 
SetZero(a);
  • Pass by value: do not modify a; only the copied parameter value changes. This is a shallow copy of the value.
void setZero(int b) { b = 0; }
setZero(a);
  • Pass by const reference:
void SetZero(const int& b) { /* b = 0 在这里是无效的 */ } 
SetZero(a); // 会报错


8. Classes and Objects (Instances) — OOP


  • If int were a class, then 42 would be an instance.
  • Objects are instances of classes.
  • Functions inside a class — member functions — are “methods.”
  • Instantiation means constructing an object and giving values to its members.
  • Data members of a class are generally private.
  • A class may have multiple constructors, but only one destructor.
  • After a complete class definition, the final closing brace must be followed by a semicolon.
  • rule of three

Basic class model:

class 类 {
public:
    返回类型 成员函数(函数参数) {
        成员变量 = 函数参数；
    }
private:
    数据类型 成员变量；
}

If object obj calls member function func(int, v) defined in class C:

C obj;
obj.func(3);

If an object accesses member variable variab:

C obj;
obj.variab = 3;

Object declaration:

IntCell m1;            // OK：默认构造
IntCell m2(12);        // OK：有参构造
IntCell* m3;           // OK：指针声明
Intcell m4();          // 这其实是函数声明，不是对象！

m3->function();  // 指针访问成员要用箭头
m1.function();

Interface–Implementation Separation

IntCell.h: header file, including the class declaration and constructor.

  • Header files may also use #pragma once to prevent repeated inclusion.
#ifndef INTCELL_H     // 防止重复包含
#define INTCELL_H

class IntCell {
public:
    IntCell(int initialValue = 0); // 构造函数声明
    int read() const;              // 读取
    void write(int x);             // 写入
private:
    int storedValue;               // 私有成员
};
#endif

IntCell.cpp：

#include "IntCell.h"

IntCell::IntCell(int initialValue)   // 类::成员函数(变量)
    : storedValue(initialValue) { }

int IntCell::read() const {
    return storedValue;
}

void IntCell::write(int x) {
    storedValue = x;
}

The this Pointer

  • The difference between a member function and an ordinary free function is that the member function implicitly receives a this pointer.
  • When an object calls a member function, the compiler implicitly passes the address of that object into the function. this is therefore a pointer to the current object.
  • this->value = v means: assign v to the member variable value of the current object.
  • -> is the operator for accessing a member through a pointer.
  • this->value is equivalent to (*this).value.
  • Since value is already a member variable, one may simply write value = v. But if the parameter and the member share the same name, this-> becomes necessary:
class Tick {
    int value;
public:
    void setValue(int value) {
        this->value = value;  // 左边是成员变量，右边是参数
    }
};
Tick b1, b2;  // instantiate `Tick`, producing objects
b1.setValue = 10;
b2.setValue = 20;
cout << b1.value << " " << b2.value; // 输出 10 20

Rule of Three

If a custom class manages resources manually, then the copy constructor, copy assignment operator, and destructor usually must be defined together.

Copy Constructor

  • Called when one object is used to initialize a new object.
class MyArray {
    int* data;
    int size;
public:
    MyArray(int s) : size(s) {data = new int[size];}

    // ✅ 拷贝构造函数
    MyArray(const MyArray& other) {
        size = other.size;
        data = new int[size];
        for (int i = 0; i < size; i++) data[i] = other.data[i];
    }
};
MyArray a1(5);
MyArray a2 = a1; // 调用运行

The copy constructor above performs a deep copy of other, ensuring that the new array has its own storage. Otherwise the destructor could free shared memory twice and crash the program.

Copy Assignment Operator

  • Called when assigning to an already existing object.
class MyArray {
    int* data;
    int size;
public:
    MyArray(int s) : size(s) { data = new int[size]; }

    // ✅ 拷贝赋值运算符
    MyArray& operator=(const MyArray& other) {
        if (this != &other) {            // 防止自赋值
            delete[] data;               // 先释放旧资源
            size = other.size;
            data = new int[size];
            for (int i = 0; i < size; i++) data[i] = other.data[i];
        }
        return *this;  // 允许链式赋值 a = b = c;
    }
};
MyArray a1(5);
MyArray a2(10);
a2 = a1; // 调用运行

This ensures that the old resource is safely released before allocating and copying the new one.

Destructor

  • Used to release resources such as heap memory.
class MyArray {
    int* data;
    int size;
public:
    MyArray(int s) : size(s) { data = new int[size]; }

    // ✅ 析构函数
    ~MyArray() {
        delete[] data;   // 释放内存
    }
};
// 对象遇到这一段自动销毁
