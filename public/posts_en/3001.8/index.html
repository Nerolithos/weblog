🧮 CSC3001 LEC17-19 function mapping

November 17, 2025

(Covers Lectures 13 and 14)

LN13


Function Mappings


  • Three kinds of mappings for functions
    Given sets A and B, a function f:A → B sends each input a ∈ A to a unique output f(a) ∈ B. If different inputs never map to the same output, then f is an injection. If every element of B is hit by some input, then f is a surjection. If both properties hold, then f is a bijection. A bijection implies |A| = |B|, which means we can transform a hard-to-count set A into an easier-to-count set B and then count |A| through |B|.
  • The “bijection counting method”
    The standard steps are:
    ① find a set B that is easier to count;
    ② construct a map f:A → B;
    ③ prove that f is a bijection (or later, a k-to-1 map);
    ④ count |B|, or use the Division Rule |A|=k|B|.


Division Rule


  • Division Rule:
    If f: A → B is k-to-1, meaning every b ∈ B is hit by exactly k elements of A, then |A| = k·|B|, and hence $|B|=\dfrac{|A|}{k}$. This generalizes the bijection case k=1.

Four Common Models


  1. Cardinality of the Power Set
    Recall from the first half of the semester: the power set of a set is the set of all its subsets. Suppose |M| = k. A: the elements of the power set
    B: k-bit binary strings, where 0 or 1 indicates whether each element of M is absent from or present in the current subset. There are 2k such binary strings. Since B covers all possible subsets, A → B is a bijection, so |A| = |B| = 2k.

    Similarly, the number of subsets of size m can be viewed through an m!·(k-m)!-to-1 mapping from permutations to combinations, which explains the formula: $ rac{k!}{m!·(k-m)!}= {k \choose m}$


  1. Circular Permutations
    A: linear permutations of n elements, so P(n) = n!

    B: circular permutations of n elements. Because a circular arrangement has no distinguished beginning or end, we must quotient out by rotational equivalence. For example, ABCD, BCDA, CDAB, and DABC represent the same circular ordering. In general, each circular ordering corresponds to n different linear orderings depending on which element is placed first. Hence A → B is n-to-1. Therefore, by the Division Rule, $|B|=\dfrac{n!}{n}=(n-1)!$.


  1. Redundancy Caused by Identical Pieces
    Consider the number of ways to place two non-attacking rooks on a chessboard. Map each arrangement to a quadruple (r(1),c(1),r(2),c(2)) with $r(1) eq r(2),c(1) eq c(2)$.
    However, swapping which rook is called “1” and which is called “2” should not produce a new arrangement. Therefore A → B is 2!-to-1: $|B|=\dfrac{|A|}{2!}=\dfrac{8\cdot8\cdot7\cdot7}{2}=1568$.

  1. A Mathematical Form of the Stars-and-Bars Method
    Suppose we choose one dozen (12) donuts from five flavors. We can encode each choice as a binary string. A: use four 1s to divide the string into five ordered segments, and fill the segments with 0s to indicate how many donuts of each flavor are chosen. The resulting 16-bit binary string represents one choice. For example, for the 5-tuple (choco, icing, plain, glazed, crème), the string 1100010000010000 represents three plain donuts, five glazed donuts, and four crème donuts. B: the set of ways to choose one dozen donuts from five flavors. Since A covers all possibilities, A → B is a bijection, so |B| = |A| = $16 \choose 4$ = 1820.

So every stars-and-bars problem can be translated into a binary-string model. Its general solution is: ${\text{number of bars} + \text{number of selections} \choose \text{number of bars}} = {C(0)+C(1) \choose C(1)}$ where C() denotes the number of occurrences of the corresponding symbol in the binary string.


Exercise


  • Twelve people are divided into three groups of equal size and seated around three tables. How many seating arrangements are there? $$ rac{12!}{(4!)^3·3!} imes((4-1)!)^3 = rac{12!·(3!)^2}{(4!)^3}=1247400$$

    $$Takeaway:\ for\ an\ ordering\ of\ n\ items\ with\ k\ unordered\ groups\ I,\ the\ number\ of\ distinct\ orderings\ is\ {n!\over \prod_k |I_k|!}. $$



Catalan Numbers


Here are four seemingly different problems:

(A) In an n-by-n square grid, how many paths are there from the lower-left corner to the upper-right corner if one may move only right or up and may never cross the diagonal?
(B) How many Dyck words of length 2n are there, i.e. strings with the same number of x’s and y’s such that in every prefix, the number of x’s is at least the number of y’s?
(C) How many valid ways are there to match n pairs of parentheses?
(D) If each step is an up-slope or down-slope segment of fixed length, how many valid mountain outlines of n up-steps and n down-steps are there?

These problems are pairwise in bijection, so they all have the same answer, namely the Catalan number.


  • Example of a bijection
    Encode the right/up path in (A) as a string with n x’s and n y’s. Requiring every prefix to have at least as many x’s as y’s is exactly the definition of a Dyck word in (B). Then interpret “(” as x and “)” as y, and we obtain the valid parenthesis strings in (C).

  • Proof by the reflection principle
    Using (A) as the model problem, first count all right/up paths from (0,0) to (n,n), namely $inom{2n}{n}$. Then construct a bijection between the paths that cross the diagonal and the paths from (0,0) to (n-1,n+1): at the first point where a path crosses the diagonal, reflect the remainder of the path by swapping every right step with an up step and every up step with a right step. This produces a corresponding path from (0,0) to (n-1,n+1). Since a path that never exceeds the diagonal can never reach (n-1,n+1), the crossing paths are in 1-to-1 correspondence with all paths from (0,0) to (n-1,n+1).

    Thus: |(0, 0)→(n, n) Exceed| = |(0, 0)→(n-1, n+1) All| = $inom{2n}{n+1}$

    Hence $$Catalan(n)=inom{2n}{n}-inom{2n}{n+1}= rac{1}{n+1}inom{2n}{n}.$$

    This proves the Catalan formula.
