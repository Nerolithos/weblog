# CSC Final



## String Processing



### `a.replace(b, c)`
- `a.replace(b, c)` replaces every occurrence of substring `b` in string `a` with `c`. You can also use it to remove `b` entirely, or to insert `c` between every two characters.

- It works for both characters and strings.

- For example, to remove the space inside a phrase: `"Hello world".replace(" ", "");`

- Application: OJ Problem P1019 (BanishLetter2) 

  Write a static method that takes a String and many char values. Delete all the characters in the String that equals to the char values, and return the result. For example: 
```shell
Input:            Output:
Hello World       He Wrd
o l k
```



- Certainly, we can inspect the string character by character and filter out the required letters:

```java
import java.util.Scanner;
public class BanishLetter2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String in = sc.nextLine();
        String filterset = sc.nextLine();
        String[] input = in.split("");
        String[] filter = filterset.split(" ");
        next:
        for (int i=0; i<input.length; i++) {
            for (int j=0; j<filter.length; j++) {
                if (filter[j].equals(input[i])) continue next;
            }
            System.out.print(input[i]);
        }
        System.out.println();
    }
}
```

- However, with `replace`, we can process the string directly without first splitting it into an array:

```java
import java.util.Scanner;
public class BanishLetter2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String input = sc.nextLine();
        String filterset = sc.nextLine();
        String[] filter = filterset.split(" ");
        for (int i=0; i<filter.length; i++) input = input.replace(filter[i], "");
        System.out.println(input);
    }
}
```





### `a.startWith(b)` and `a.endWith(c)`

- They test whether string `a` starts with substring `b`, or ends with substring `c`.

- Application: OJ Problem P1015 (Longest Common Prefix)

  Write a static method that takes a String array and return the longest common prefix. Return an empty String if common prefix does not exist.

  Input1: `flower flow flight`

  Output1: `fl`

  Input2: `hello world csc1003`

  Output2:



- Idea: assume the first word is the current LCP. If any later word does not `startsWith` it, remove the last character from the candidate prefix. Repeat until every remaining word matches the trimmed LCP.

```java
import java.util.Scanner;

public class LCP {
    public static String lcp(String[] strs) {
        String prefix = strs[0];  // Assume the first word is the longest common prefix initially
        for (int i = 1; i < strs.length; i++) {
            while (!strs[i].startsWith(prefix)) {  // If a word does not match the current prefix, shorten it by one character
                prefix = prefix.substring(0, prefix.length() - 1);
                if (prefix.isEmpty()) {
                    return "";
                }
            }
        }
        return prefix;  // Once all words match the trimmed first word, return it
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String input = sc.nextLine();
        String[] words = input.split(" ");
        String result = lcp(words);
        System.out.println(result);
    }
}
```





### `a.substring(b, c)`

- `a.substring(b, c)` extracts the characters of string `a` whose indices lie in `[b, c)`. It can also be used to isolate a single character such as `a[b]`.

- For example, print a string one character at a time:

  ```java
  String str = "Hello World"
  for (i=0; i=str.length; i++) {
    if (!str.substring(i, i+1).equals(" ")) {
      System.out.println(str.substring(i, i+1)
    }
  }
  ```

- Application: OJ Problem P1021 (Bool) 

  A Boolean expression is an expression that evaluates to either true or false. Valid expressions follow the following conventions:

  the result of 't' is true; the result of 'f' is false; '!(subExpr)' means to perform logical NOT operation on the internal expression subExpr; '&(subExpr1, subExpr2, ..., subExprn)' means to perform logical AND operation on 2 or more internal expressions subExpr1, subExpr2, ..., subExprn; '|(subExpr1, subExpr2, ..., subExprn)' means to perform logical OR operation on 2 or more internal expressions subExpr1, subExpr2, ..., subExprn.

  Given a boolean expression in the form of a String, you need to return the result of the expression.

  ```shell
  Input1: |(f,f,f,t)           Input2: !(&(f,t))
  
  Output1: t                   Output2: true
  ```



- Converting the string into an array and then looping over it would be unnecessarily cumbersome here. By contrast, `substring` can directly select part of the string, whereas arrays would require extra loops, possibly nested ones.

- My solution is this: when the method detects an operator (`&` or `|`), it removes the surrounding parentheses and commas, then recursively evaluates the resulting string; when it sees `"t"` or `"f"`, it returns the corresponding Boolean value; when it sees `"!"`, it returns the negation of what follows.

```java
import java.util.*;

public class Bool {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String expression = sc.nextLine();
        System.out.println(evaluateExpr(expression));
    }

    public static boolean evaluateExpr(String expr) {
        // Handle the base cases `f` and `t` inside the expression 
        if (expr.equals("t")) return true;
        if (expr.equals("f")) return false;
        // Every `return` sends information back to the caller, so in recursion it does not prematurely terminate the overall computation
        
      if (expr.startsWith("!")) {
            return !evaluateExpr(expr.substring(2, expr.length() - 1)); // Remove the outer parentheses
        }

        if (expr.startsWith("&")) {
            String innerExprs = filter(expr.substring(2, expr.length() - 1));
            for (int i=0; i<innerExprs.length(); i++) {
                if (!evaluateExpr(innerExprs.substring(i, i+1))) return false;
                // For AND, a single `f` makes the whole result false
            }
            return true;
        }

        if (expr.startsWith("|")) {
            String innerExprs = filter(expr.substring(2, expr.length() - 1));
            for (int i=0; i<innerExprs.length(); i++) {
                if (evaluateExpr(innerExprs.substring(i, i+1))) return true;
                // For OR, a single `t` makes the whole result true
            }
            return false;
        }
        return true; // This return value does not matter, because valid input should never let the program reach this line
    }
    public static String filter(String expr) {
        String split = expr.replace(",", "");
        return split;
    }
}
```





### `a.contains(Char b)`

- It checks whether string `a` contains character `b`, returning `true` or `false`. It is commonly used as a condition in `if` or `while` statements.





### `a.IndexOf(Char b)` and `a.IndexOf(String b)`

- This is similar to the previous one, but it can locate either a character or a string. It returns the index where `b` first appears in `a`, as an `int`.
- If you only want to test whether `a` contains `b`, simply check whether the return value is `-1`.







## Common Pitfalls



### `a.length` 和 `a.length()`

- `a.length` is used when `a` is an array, whereas `a.length()` is used when `a` is a string. They cannot be mixed up.
- Arrays and strings are both data types, but their interfaces differ: an array’s length is a field, while a string’s length is obtained through a method.




### `(<type>) b` 和 `Integer.parseInt(b)`
- Explicit casting `type a = (type) b` is mainly used among numeric types (`long`, `short`, `int`, etc.), and is commonly used for truncation or narrowing conversions.



### On `return` and Recursive Methods

- Many beginners think `return` simply ends a method and jumps back to the call site. In recursion, however, `return` sends control back to the previous recursive frame rather than ending the entire recursive process immediately.
- For example, in a method such as `recur(String a)`, you may write `return recur(String b) + recur(String c)` to perform divide-and-conquer style recursion, such as decomposing an expression by operators and solving it piece by piece.







## OOP and DT

- Whenever we see `Class.b` in Java, it indicates that `b` is a member field of class `Class`. For example, if we define a class `Coffee`, declare `public final foam;` inside it, and set `this.foam = foam;` inside the constructor `public Coffee(double foam)`, then `foam` becomes an accessible field:

  ```java
  Coffee cappuccino = new Coffee(1/3);  // This line is precisely calling the constructor!
  double foampct = cappuccino.foam  // foampct = 0.333333...
  ```

- A more common example is `array.length`, where `length` is an immutable field of the given array.
- Note: if a member field is not `final` (i.e., it is mutable), it is usually better not to access it directly. Instead, use getter methods so that the class preserves proper encapsulation.



- As for constructing a DT, here is the example from the lecture:

  ```java
  public class Complex {
      private final double re;
      private final double im;
  
      public Complex(double real, double imag) {
          this.re = real;
          this.im = imag;
      }
  
      public Complex plus(Complex b) {
          double real = this.re + b.re;
          double imag = this.im + b.im;
          return new Complex(real, imag);
      }
  
      public Complex times(Complex b) {
          double real = this.re * b.re - this.im * b.im;
          double imag = this.re * b.im + this.im * b.re;
          return new Complex(real, imag);
      }
  
      public double modulus() {
          return Math.sqrt(re * re + im * im);
      }
  
      // Test code
      public static void main(String[] args) {
          Complex a = new Complex(3.0, 4.0);
          Complex b = new Complex(1.0, 2.0);
          System.out.println("a: " + a);
          System.out.println("b: " + b);
          System.out.println("a + b: " + a.plus(b));
          System.out.println("a * b: " + a.times(b));
          System.out.println("Modulus of a: " + a.modulus());
      }
  }
  ```

  - This program can represent complex numbers and compute their addition, multiplication, and modulus.
  - Every time a complex number is created, the constructor is invoked.
  - In `a.plus(b)` and `a.times(b)`, the argument `b` cannot be omitted, because the parameter named `b` in the method declaration is not the same thing as the actual object passed in during invocation.
